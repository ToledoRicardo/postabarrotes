# Imágenes de Billetes

Este directorio está reservado para las imágenes de los billetes mexicanos.

## Billetes necesarios:
- billete_20.png
- billete_50.png
- billete_100.png
- billete_200.png
- billete_500.png
- billete_1000.png

Por ahora, el sistema muestra un icono de dinero con el valor del billete. Puedes agregar las imágenes reales de los billetes aquí y actualizar el código en `punto_venta_screen.dart` para mostrarlas.

## Tamaño recomendado:
- Ancho: 300px
- Alto: 150px
- Formato: PNG con fondo transparente
